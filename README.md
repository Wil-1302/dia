# Un lugar para descansar 🥑🌙

Una pequeña experiencia web interactiva: un **cuadro vivo** inspirado (no copiado) en la
noche estrellada, hecho con mucho cariño. Explora la pintura, toca todo lo que veas y
encuentra las **10 paltitas escondidas**. En el camino aparecen mensajes, momentos
especiales y una carta final.

## ✨ Qué incluye
- Intro con estrellas que aparecen, texto tipo máquina de escribir y una flor para comenzar.
- Cielo animado en **Canvas**: pinceladas vivas, estrellas que parpadean, luna que ilumina,
  nubes, viento, partículas, estrellas fugaces y parallax.
- Paisaje interactivo hecho en **SVG**: árboles, casitas, piedras, arbustos, flores,
  mariposas y pájaros. Casi todo reacciona al tocarlo.
- 10 paltitas ocultas, cada una con su propia personalidad y animación (todas en SVG).
- Recompensas con tarjeta flotante (GIF + frase romántica) y +40 mensajes originales.
- Momentos especiales tras la 3ª, 5ª, 7ª y 9ª palta, y un gran final tras la 10ª.
- Música y sonidos **100% procedurales** (Web Audio API), con botón para activar/silenciar.
  No hay reproducción automática.

## 🛠️ Tecnología
HTML + CSS + JavaScript (ES6). Sin frameworks. Canvas + SVG. Todo autocontenido.

## ▶️ Cómo verlo
Está listo para **GitHub Pages**. También puedes abrir `index.html` directamente,
o servirlo localmente:

```bash
python3 -m http.server 8080
# luego abre http://localhost:8080
```
